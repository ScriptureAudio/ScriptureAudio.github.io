Psalm 139, KJV

**Version 3, Upload date 11/10/17 **Download an MP3 of Psalm 139, Version 3 here. **Download an MP3 of Psalm 139, Music Only here.

So with this final Version 3 of Psalm 139, this concludes our 6 months of memorizing Psalms 15, 103 and 139! A music-only Version 4 to enjoy is linked here.

As a thank-you for sticking with us, I’d like to offer you all a full set of memorization tracks of Psalm 23, set to Felix Mendelssohn’s Piano Trio #1 in D Major, 2nd mv’t, for your little ones who’ve not yet learned . There are only three versions of this psalm, since it’s rather simple, but please enjoy the tracks and share with others if you like!

I’ll be making these memorization tracks, as well as the entirety of the recording project from March, available as a CD for purchase, so please keep an eye out. I’ve been hearing that the convenience of a CD might make memorization easier so if internet access or confusing instructions have been a problem, perhaps a CD will be a benefit to you?

As I mentioned back in June, this was a test project to gauge the level of your interest and enthusiasm for a method of learning Scripture that has benefited my family. If you have comments or suggestions, please pass it along at scriptureaudio@gmail.com.

Thanks again to Scott and the Centerville String Quartet for their efforts helping with this project, they’ve been great to work with. I’ve had a lot of fun and looking forward to doing this again.

Jonathon
